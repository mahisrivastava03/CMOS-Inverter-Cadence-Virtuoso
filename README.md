# CMOS-Inverter-Cadence-Virtuoso
This project demonstrates the complete custom IC design flow of a CMOS Inverter using Cadence Virtuoso as part of my STPVD Summer Training at IIIT Allahabad.
## Tools Used

- Cadence Virtuoso
- Spectre Simulator
- Assura/PVS (DRC & LVS)

## Design Flow

- CMOS Inverter Schematic
- Symbol Creation
- Testbench Design
- Pre-layout Simulation
- Layout Design
- DRC Verification
- LVS Verification
- Post-layout Simulation

## Results

The inverter successfully passed DRC and LVS verification.

Transient simulations before and after layout confirmed correct inverter functionality.

## Learning Outcomes

- CMOS Design Fundamentals
- Custom IC Layout
- DRC & LVS Verification
- Layout versus Schematic Matching
- Pre-layout and Post-layout Simulation
